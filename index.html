How To Play

Objective:
Cover the whole grid by eating every red dot and growing the snake.

Controls:
Use the arrow keys to move.
Press Pause or Space bar to pause and resume.

Rules:
Do not hit the wall.
Do not hit your own body.
Each red dot gives 1 point.
Every 10 points uses a different sound.

Continue:
A continue takes you back to the game state from before the last red dot was eaten.
Then the game shows a 3 second countdown and resumes automatically from that exact same state.
Continue can be used by watching an ad, or by spending a key.

Redo (from Pause popup):
Press Pause (or Space bar) to open the pause popup, then press the ↶ Redo button.
Redo also returns to the state from before the last red dot was eaten, with the red dot back in the same place.
You can press redo repeatedly to go back eat-by-eat until your first eat of the game.
Ad redos per game are the same count as ad continues for that board size.
Keys are shared power: the same keys can be used for both Continue and Redo.
You can buy keys anytime, and buying keys gives more uses for both Continue and Redo.
